# Phase 1: Core Data Analyst Skill Resources

> Curated free learning resources for the 6 most-requested data analyst skills in NZ job listings.  
> All URLs verified June 2026. Mix of interactive, article, video, course, and documentation formats.

## 1. SQL Basics

_SELECT, WHERE, GROUP BY, JOINs, subqueries, CTEs — PostgreSQL-focused with interactive practice._

| Resource | URL | Type | Why It's Good | Approx. Time |
|----------|-----|------|---------------|-------------|
| **SQLBolt** | [https://sqlbolt.com/](https://sqlbolt.com/) | interactive | Best zero-friction start: 20 interactive lessons in the browser running real SQL — no setup, no signup. Covers SELECT through table creation. | 4–6 hours |
| **PostgreSQL Tutorial** | [https://www.postgresqltutorial.com/](https://www.postgresqltutorial.com/) | tutorial / article | The most comprehensive free PostgreSQL resource; practical examples from basic querying through CTEs, with downloadable sample databases. | 15–20 hours |
| **Select Star SQL** | [https://selectstarsql.com/](https://selectstarsql.com/) | interactive book | Project-based interactive book using real-world death-penalty data; teaches a mental model for _thinking_ in SQL, not just memorising syntax. | 5–6 hours |
| **DataCamp: Introduction to SQL** | [https://www.datacamp.com/courses/introduction-to-sql](https://www.datacamp.com/courses/introduction-to-sql) | course (video + exercises) | Free-tier course with bite-sized videos and in-browser coding exercises; 4.8 stars from 54,000+ reviews. Covers both PostgreSQL and SQL Server differences. | ~2 hours |

## 2. Advanced SQL

_Window functions (ROW_NUMBER, RANK, LAG, LEAD), recursive CTEs, query optimisation (EXPLAIN, indexes), complex joins._

| Resource | URL | Type | Why It's Good | Approx. Time |
|----------|-----|------|---------------|-------------|
| **PostgreSQL Tutorial: Window Functions** | [https://www.postgresqltutorial.com/postgresql-window-function/](https://www.postgresqltutorial.com/postgresql-window-function/) | article | Complete reference for all PostgreSQL window functions (ROW_NUMBER, RANK, DENSE_RANK, LAG, LEAD, FIRST_VALUE, NTILE) with worked examples and frame-clause explanation. | 3–4 hours |
| **PostgreSQL Exercises** | [https://pgexercises.com/](https://pgexercises.com/) | interactive | Progressive exercises from basic SELECT to recursive queries and window functions, all on a single realistic dataset. Includes a dedicated "Recursive" category. | 8–12 hours |
| **Use The Index, Luke** | [https://use-the-index-luke.com/](https://use-the-index-luke.com/) | article / book | The definitive free guide to SQL indexing and query performance; covers EXPLAIN, B-tree anatomy, join strategies, and tuning across PostgreSQL, MySQL, Oracle, and SQL Server. | 6–8 hours |
| **Official PostgreSQL Tutorial** | [https://www.postgresql.org/docs/current/tutorial.html](https://www.postgresql.org/docs/current/tutorial.html) | documentation | The authoritative source — Chapter 3 covers advanced features including window functions (sec. 3.5), views, foreign keys, and transactions. Comes straight from the PostgreSQL team. | 4–6 hours |
